# SRTLE Mock Exam - محاكي اختبار الهيئة

تطبيق ويب تفاعلي (SPA) بالكامل لمراجعة ومحاكاة اختبار الهيئة السعودية للتخصصات الصحية (SRTLE) لأخصائيي الأشعة والتكنولوجيا الطبية.

## 🚀 المميزات
- **4,192 سؤال وجواب** مستخرجة بدقة كاملة من ملف الأسئلة الرسمي.
- **تصفح حسب الفصول**: إمكانية دراسة وحل أسئلة كل فصل بشكل مستقل.
- **محاكي الاختبار**: 200 سؤال عشوائي بمؤقت زمني يحاكي الاختبار الفعلي.
- **نظام التقييم الفعلي**: يحسب النتيجة من 200 إلى 800 (درجة النجاح 530).
- **وضع التدريب (Practice Mode)**: تغذية راجعة فورية تظهر الإجابة الصحيحة أو الخاطئة بمجرد النقر على الخيار.
- **حفظ تلقائي (Auto-save)**: حفظ التقدم والنتيجة الأخيرة تلقائياً في المتصفح (`LocalStorage`).
- **دعم المظهر الداكن والفاتح (Dark/Light Mode)**.

## 🛠️ التقنيات المستخدمة
- HTML5 / CSS3 (Vanilla)
- JavaScript (ES6 Modules)
- لا يتطلب أي قواعد بيانات أو سيرفرات معقدة (يعمل مباشرة على المتصفح عبر قراءة `questions.json`).

## ⚡ التشغيل والرفع على Vercel
المشروع جاهز تماماً للتشغيل والرفع على **Vercel** مباشرة وبدون أي إعدادات إضافية:
1. قم بربط مستودع GitHub هذا بحسابك في Vercel.
2. سيقوم Vercel باكتشاف ملف `index.html` تلقائياً ونشر الموقع في ثوانٍ معدودة.

أو للتشغيل محلياً:
- يمكنك فتح ملف `index.html` مباشرة باستخدام أي خادم محلي (مثل Live Server في VS Code أو `python3 -m http.server`).
